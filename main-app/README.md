# main-app

## Installation

```shell
$ yarn
```

## How to use

Run the following commands in the root directory.

```bash
yarn start
```

相应的联邦模块 `component-app` 也需要用以下任一命令启动
```bash
yarn start
```

或者 

```bash
yarn webpackServe
```

Runs the app in the development mode.\
Paste [http://localhost:3001](http://localhost:3001) to view it in the browser.
