# component-app

## Installation

```shell
$ yarn
```

## How to use

### 单独运行调试本项目
Run the following commands in the root directory.

```bash
yarn dev
```

Runs the app in the development mode.\
Paste [http://localhost:3002](http://localhost:3002) to view it in the browser.

### 模块联邦调试
与 `main-app` 项目进行模块联邦调试
Run the following commands in the root directory.

```bash
yarn start
```

或者 

```bash
yarn webpackServe
```
